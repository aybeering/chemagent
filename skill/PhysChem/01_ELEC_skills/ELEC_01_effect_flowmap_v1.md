# ELEC_01_effect_flowmap_v1 — 电子效应"推/拉从哪里来、怎么传"的总路由

## Triggers
- 需要对"某取代基对某位点/骨架"的**电子推/拉方向、传递路径、衰减与主导机制**做定性判断  
- 需要把电子效应分解成 **±I / ±M(±R) / 超共轭 / 场效应 / 构象共轭开关**并给出合成结论

## Inputs
- 分子表示：SMILES / 结构描述 / 官能团簇列表  
- 关注位点：原子编号或"官能团中心/反应中心"的文字描述  
- 环境假设（可选）：溶剂极性、是否强电场/界面、是否带电、是否存在配位/氢键锁定构象  
- 需要输出的尺度（可选）：局部（1–3键范围）/ 全局（跨共轭体系）

## Outputs
- `effect_summary`: 总结（哪类效应主导、净推/拉、作用范围）  
- `channels`: 分通道结论（I/M/H/FIELD/CONF）及置信度  
- `transmission_path`: 电子效应传播路径图（σ链/π网络/空间近邻）  
- `dominant_sites`: 最可能被增强亲电/亲核/氧化/还原敏感的位点列表（定性）

## Rules
- **先判"能不能传"**：σ 一定能传但衰减快；π 需要共轭连续与轨道匹配；场效应靠空间距离与偶极取向；超共轭需要合适的 σ→π/σ→p 供给几何。  
- **先判"通道冲突"**：−I 与 +M 可并存（如卤素）；以"对目标位点的有效耦合"为准，而非取代基本性格。  
- **先判"构象是否切断共轭"**：共面性不足时，±M显著降权。  
- 默认优先级（无额外信息时）：**共轭连续的 ±M > 强 −I（近端）> 超共轭 > 场效应（近端/界面增强）**；若界面强电场/带电基团近邻，则场效应上调优先级。

## Steps
1. 识别目标位点与关键取代基（电子源/电子汇候选）。  
2. 构建传播通道：  
   - `σ-path`：目标位点与取代基间的最短键路径（键数）  
   - `π-path`：是否处于同一共轭网络（芳环/烯/羰/腈等）  
   - `space-field`：空间距离近邻偶极/电荷/氢键方向  
3. 分别调用子技能卡：`ELEC_02_I_sigma_v1`、`ELEC_03_M_pi_v1`、`ELEC_04_hyperconj_v1`、`ELEC_05_field_v1`、`ELEC_06_conformation_switch_v1`。  
4. 汇总：给出主导通道、净推/拉、作用范围、对局部反应性与前线轨道（HOMO/LUMO）的大方向影响（仅定性）。  
5. 输出可读的"结论+依据清单"（每条依据绑定通道与结构证据）。

## Examples
- 芳环 para-OMe：π网络连续 → +M 强；σ 也有 −I 但次要；净给电子，环上亲电取代更易。  
- CF₃ 连在饱和链上：无 π 通道 → 主要 −I，经 σ 传递随键数衰减；近端位点更受影响。

## Guardrails
- 不输出定量（Hammett 常数、轨道能量数值、pKa 等）除非上游提供。  
- 不把"取代基性格"硬套到所有位点：必须显式给出"对目标位点的耦合路径"。  
- 遇到可能互变/强配位/显著电荷重排体系：降低置信度并标注"需额外信息"。
